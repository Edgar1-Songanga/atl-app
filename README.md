🚀 ATL Smart

Plataforma digital para gestão de ATL (Atividades de Tempos Livres), com monitoramento de crianças, comunicação com pais e sistema inteligente de pagamentos.

💡 Funcionalidades

- Dashboard do encarregado
- Perfil da criança
- Check-in digital
- Pagamentos com divisão automática

🧠 Diferencial

Sistema que divide automaticamente:

- Comissão da plataforma
- Receita do ATL

▶️ Como usar

Abrir "index.html" no navegador.

🌍 Visão

Transformar-se numa plataforma SaaS líder em África para gestão infantil.
